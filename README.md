# 🤖 AI RAG Chatbot using Endee

## Overview
This project is a Retrieval Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions.

## Features
- PDF Upload
- Semantic Search using embeddings
- RAG-based answer generation
- Streamlit web app

## How it works
1. Upload PDF
2. Convert text into embeddings
3. Store in vector database
4. Retrieve relevant chunks
5. Generate answer

## Setup
---
-pip install -r rag_chatbot/requirements.txt
-streamlit run rag_chatbot/app.py

## Use Case
Document Q&A system using AI
